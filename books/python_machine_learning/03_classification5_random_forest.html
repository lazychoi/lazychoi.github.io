03장 사이킷런 - Random Forest

Published

November 9, 2022

알고리즘

  1. n개의 랜덤한 부트스트랩 샘플 뽑음(중복 허용)
  2. 샘플에서 결정 트리 학습
    1. 중복 허용하지 않고 랜덤하게 d개의 특성 선택
    2. 노드 분할
  3. 앞의 두 단계를 k번 반복
  4. 각 트리의 예측을 모아 다수결 투표로 클래스 레이블 할당

특징

  • 하이퍼파라미터 튜닝에 많은 노력을 기울일 필요 없다. 일반적으로 가지치기할 필요도 없다.
  • 부트스트랩 샘플 크기가 작아지면 -> 과대적합 영향 줄고 성능도 줄어듦
  • 부트스트랩 샘플 크기 = 훈련 데이터셋의 샘플 개수 => 균형 잡힌 편향-분산 트레이드오프
  • 분할에 사용할 특성 개수 \(d = \sqrt{m}\) (m은 훈련 데이터셋의 특성 개수)
from matplotlib.colors import ListedColormap
import matplotlib.pyplot as plt
import warnings
warnings.filterwarnings(action='ignore')

def plot_decision_regions(X, y, classifier, test_idx=None, resolution=0.02):

    markers = ('s', 'x', 'o', '^', 'V')
    colors = ('red', 'blue', 'lightgreen', 'gray', 'cyan')
    cmap = ListedColormap(colors[:len(np.unique(y))])

    x1_min, x1_max = X[:, 0].min() -1, X[:, 0].max() + 1
    x2_min, x2_max = X[:, 1].min() -1, X[:, 1].max() + 1
    xx1, xx2 = np.meshgrid(np.arange(x1_min, x1_max, resolution),
                           np.arange(x2_min, x2_max, resolution))
    Z = classifier.predict(np.array([xx1.ravel(), xx2.ravel()]).T)
    Z = Z.reshape(xx1.shape)
    plt.contourf(xx1, xx2, Z, alpha=0.3, cmap=cmap)
    plt.xlim(xx1.min(), xx1.max())
    plt.ylim(xx2.min(), xx2.max())

    for idx, cl in enumerate(np.unique(y)):
        plt.scatter(x=X[y == cl, 0], y=X[y == cl, 1],
                    alpha=0.8, c=colors[idx],
                    marker=markers[idx], label=cl,
                    edgecolor='black')
    if test_idx:
        X_test, y_test = X[test_idx, :], y[test_idx]

        plt.scatter(X_test[:, 0], X_test[:, 1],
                    facecolors='None', edgecolor='black', alpha=1.0,
                    linewidth=1, marker='o', s=100, label='test set')

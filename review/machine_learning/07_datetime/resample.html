1 시계열 데이터 resample

  • closed=‘right’ => ( 3:00, 6:00 ] or 3:00 < x <= 6:00
  • closed=‘left’ => [ 3:00, 6:00 ) or 3:00 <= x < 6:00
  • label=‘left’ => 왼쪽 표시
  • label=‘right’ => 오른쪽 표시

출처

time_index = pd.date_range('1/1/2000', periods=9, freq='T')
df = pd.DataFrame(index=time_index, data=range(9))
df

df.resample('3T').sum()

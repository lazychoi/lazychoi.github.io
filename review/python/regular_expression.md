## html 페이지에서 태그 삭제: `<([^>]+)>`

꺽쇠 괄호와 꺽쇠 괄호로 둘러싸인 문자열을 삭제한다.

- 찾을 문자열이 왼쪽 꺽쇠("<")로 시작하고 오른쪽 꺽쇠(">")로 끝난다.
- 대괄호 안 캐럿("[^]")은 캐럿 대괄호 안의 문자를 제외한 문자를 가리킨다.
- 괄호("()")는 특정 패턴을 묶어서 반복 기호와 함께 사용한다.
- 플러스("+") 기호는 바로 앞의 문자가 1번 이상 반복되는 것을 뜻한다.

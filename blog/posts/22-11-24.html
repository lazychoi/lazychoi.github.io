챗봇 프로젝트

조원들에게 보낸 메시지 1. 프로젝트에서 작업할 폴더 만들기 2. 작업할 폴더 안에 들어가서 git init 3. git remote add origin git@github.com:doksan-tech/chatbot.git 4. git pull origin main 5. git branch <브랜치명> ← 앞으로 자신이 사용할 브랜치명을 지으세요. 6. git checkout <브랜치명>

요렇게 기본 설정을 해서 각자 브랜치에서 작업을 하고 마지막에 PR을 해서 main으로 합치고 README 파일을 작성하면 될 것 같아요

제 개인 깃허브가 아니고 조직 깃허브예요. 모든 조원에게 admin 권한을 주었으니 모두의 권한이 똑같아요. 그래서 취업하실 때에도 사용하실 수 있을 거예요

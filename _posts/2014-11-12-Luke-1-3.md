### 1:3

**Therefore, since I myself have carefully investigated everything from the beginning, it seemed good also to me to write an orderly account for you, most excellent Theopilus.**

NIV로만 이해가 어려워 [Holynet](http://holybible.or.kr)에서 여러 성경을 비교해 읽는다. 몇몇 단어에 링크가 걸려있는게 눈에 띄었다. 클릭해 보니 단어에 대한 해설이 나타나는데, [한국컴퓨터선교회](http://kcm.kr)에서 만든 성경사전의 해설이었다. 좋은 도구다. 특히 [성경 도표](http://kcm.kr/dic_list.php?page=1&key=&jamo=&kword=%5B%BC%BA%B0%E6%B5%B5%C7%A5%5D)는 그림이 곁들여 있어 이해에 도움이 될뿐만 아니라 흥미롭다.

#### 문장 분석

Therefore, // since // I myself // have carefully investigated everything // from the beginning, // it seemed good // also to me // to write an orderly account // for you, most excellent Theopilus.

그러므로, // 때문에 // 내가 직접 // 주의깊게 모든 것을 조사해왔기 // 처음부터, // 그것은 좋아 보였다 // 내게도 // 순서대로 기록하는 것이 // 가장 뛰어난 당신 테오필로스를 위해


- "from the beginning"은 앞의 "from the first"와 짝을 이루는 표현인 것 같다. first는 예수님을 직접 본 사람에 대한 표현이고, beginning은 예수님의 생애 처음부터 자세히 조사했다는 의미인 듯하다.
- "most excellent"는 KCM 사전에 의하면 높은 지위에 있는 사람을 부르는 칭호다. 우리말 성경에선 "각하"로 번역되어 있는데, 우리에겐 주로 대통령에게만 붙이는 호칭이라 그다지 어울리는 것 같지는 않다. 가톨릭 성경의 번역인 "귀하"가 좀 더 적절해 보인다.
- Theopilus는 누구일까? "데오빌로", "데오필로"로 옮기는데, [위키피디아](http://en.wikipedia.org/wiki/Theophilus_(biblical))에 따르면 그가 누구인지 정확히 모른다. 그리스어로 뜻은 "friend of God", "be loved by God", "loving God"이다. 알렉산드리아의 유대인이라고도 하고, 로마 관리를 가리키는 "excellent"란 단어를 근거로 로마 관리라고도 추측한다. 혹은 유대교 사제라고 주장한다. 누가복음과 사도행전은 Theopilus에게 보내는 글이다.  

#### 성경 번역

존귀하신 테오필로스 님, 이 모든 일을 처음부터 자세히 살펴본 저도 귀하께 순서대로 적어 드리는 것이 좋겠다고 생각하였습니다.

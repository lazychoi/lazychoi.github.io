## 포스트 목록 나열하기

루트 디렉토리의 index.html에 전체 글 목록을 나타내 보자. [지킬 문서](http://jekyllrb-ko.github.io/docs/posts/)에 따르면 아래처럼 &#123;% for %&#125;문을 사용하여 문서 목록을 표시한다. 


    {%raw%}
    <ul>
      {% for post in site.posts %}
      <li>
        <a href="{{ post.url }}">{{ post.title }}</a>
      </li>
      {% endfor %}
    </ul>
    {%endraw%}

파이썬의 `for`문과 유사하다. `site.posts`는 블로그의 전체 글을 가리킬테고, `post`는 각각의 블로그 포스트를 말하는 것일게다. post에는 title 속성과 url 속성이 있는 것 같다. &#123;&#123; post.title &#125;&#125;은 각 문서의 제목을 나타내라는 것이고, 제목을 누르면 해당 글로 이동하도록 <a href="&#123;&#123;post.url"&#125;&#125;>을 추가했다. <ul></ul>태그를 써서 포스트 제목을 목록으로 표시한다.

## 날짜와 제목 표시하기

포스트 제목 앞에 포스트를 만든 날짜도 표시해보자. [리퀴드 컨닝 페이퍼](http://cheat.markdunkley.com)를 보면 사용할 수 있는 명령어와 옵션이 잘 정리되어 있다. `|` 문자는 리퀴드 필터다. 

{%raw%}
    <ul>
      {% for post in site.posts %}
        <li>
          {{ post.date | date: "[ %Y. %m. %d. ]" }}
          <a href="{{ post.url }}">{{ post.title }}</a>
        </li>
      {% endfor %}
    </ul>
{%endraw%}

- `post.date | date: "[ %Y. %m. %d. ]`는 포스트의 날짜 속성을 [연. 월. 일.]로 표시하라는 명령이다.
